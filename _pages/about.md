Hi there! 👋 I'm **Aufa Riduan Cahyadi** — a Network Technician, sysadmin, and homelab and selfhost enthusiast based in Banjarmasin, Indonesia. I spend my days building and maintaining systems that *just work*, and my nights running my own mini data center at home.

I genuinely believe that the best way to understand infrastructure is to break it, fix it, and automate it — preferably all three in the same evening.

---

## 🏢 What I Do at Work

I work at **STIKES ISFI Banjarmasin**, a private health sciences university, where I wear two hats:

**As a Programmer & SysAdmin**, I'm responsible for:

- Designing and maintaining the campus network infrastructure using **Ubiquiti UniFi** gear (UAP-LR, UAP-AC-Lite, U7 Lite, U6+, U6-LR, and more)
- Self-hosting a **UniFi OS Controller** on Proxmox for centralized WiFi management
- Building and maintaining internal **web applications** using PHP, Go, and MySQL
- Managing a **Mikrotik CCR** router as the campus core routing backbone — including firewall rules, NAT, VLANs, and traffic shaping for hundreds of users
- Keeping servers up and services running reliably, day and night

## 🖥️ The Homelab

Outside of work, I run a small but capable homelab — my personal learning lab where I experiment with infrastructure, test new tools, and self-host services I actually use every day.

The setup runs on **two Proxmox VE nodes** (`pve` and `pve2`), spanning LXC containers, full VMs, ZFS storage pools with RAID1, and a Software-Defined Networking layer for clean internal routing.

**Services I self-host:**

| Service | Role |
|---|---|
| 🎬 Jellyfin | Personal media server — my own Netflix |
| 📊 LibreNMS | Network-wide monitoring and alerting |
| 🔄 n8n | Visual workflow automation (self-hosted Zapier) |
| 🐳 Dockge | Docker Compose stack manager with a beautiful UI |
| 📈 Uptime Kuma | Real-time uptime monitoring for all services |
| 🌐 OpenSpeedTest | Self-hosted LAN & WAN speed testing |
| 🧰 OmniTools | All-in-one IT utility toolbox — no cloud needed |
| 📄 BentoPDF | Privacy-first, in-browser PDF toolkit |
| 🎵 Yubal | Self-hosted YouTube Music downloader |
| 📡 UniFi Controller | Centralized management for campus + home WiFi |

I've also recently been exploring **local AI** — running models like Gemma 4, Qwen 3.5, and Qwen 2.5 Coder on my MacBook Air M4 using LM Studio. The M4's unified memory architecture makes running 12B+ parameter models surprisingly smooth, even without active cooling.
